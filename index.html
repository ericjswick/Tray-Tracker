Firebase Configuration Required

Please configure Firebase before using this application. See the configuration steps below.

Firebase Setup Steps:
  1. Go to Firebase Console
  2. Create a new project named "sibone-tray-tracker"
  3. Enable Authentication > Sign-in method > Email/Password
  4. Create Firestore Database in production mode
  5. Enable Storage
  6. Copy your config and replace the firebaseConfig object in js/config.js
  7. Update Firestore Security Rules (see code comments)
